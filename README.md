# PostIt App
### Homework 4 Tasks link --> https://sites.google.com/view/wad20-hw4/
### Using Vue.js frontend and Node.js/Express backend
## Setup

Backend uses embedded `sqlite3` database

## To Run Project

### Backend
`cd backend`

`npm install`

`npm run db:init (Only during the first run)`

`npm start`
    
### Frontend

`cd frontend`

`npm install`

`npm run serve`
